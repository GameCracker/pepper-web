About Pepper

Pepper is building wearable Augmented Reality based computer and software, with extended sensors including EEG for brainwaves, accelerometers, infrared light, etc.

Pepper is developing native Linux and web-based software using signal processing and machine learning hacks using input from user behavior and extended sensors.

Pepper is using computer visual hacks to extend the ways people interacting with environments and each other.

Pepper is based on Linux!

Pepper is supporting and building live streaming video for virtual reality applications.

Pepper is developing open source software platform, brain-computer interface training protocals and API for developers, hobbyist and all users!
